Cmdline Efficiency

  1. Terminal emulator: iterm2.
  2. Remote server access: et/ssh.
  3. Shell: zsh/bash.
  4. Unix tools: less, head, tail, sort, uniq, comm, column, paste, grep, sed, awk, cut, xargs, find, watch, parallel, tee.
  5. Terminal multiplexer: tmux.
  6. Editor: Vim, Emacs.
